# ВБС Прокект

OWL онтологија и knowledge graph за македонски дијалектен вокабулар, изграден од транскрибирани говорни податоци.

## Опфат

Корпусот содржи **43 видеа** од **12 македонски дијалекти**, со **~5,300 транскрибирани сегменти** од кои се извлечени зборовите. Онтологијата ги моделира **зборовите (лексемите) и нивните меѓудијалектни варијанти**, поврзани со дијалектната хиерархија.

### Дијалектна хиерархија

```
Наречје (4) -> Дијалектна Група (8) -> Дијалект (24, 12 со податоци)
├── Источно: кочански, штипски
├── Западно: битолски, прилепски, кичевско-поречки, велешки, охридски, струшки
├── Северно: скопски, кумановски, овчеполски
└── Јужно: гевгелиски
```

## Брз почеток

```bash
# Инсталација
pip install -r requirements.txt

# Генерирање на knowledge graph
python scripts/extract_triples.py

# Валидација со SHACL
python scripts/validate.py

# Извршување на SPARQL queries
python queries/run_queries.py

# Вчитување во Neo4j (Aura) за визуелно истражување
cp .env.example .env      # внеси ги Aura URI/корисник/лозинка
python scripts/load_neo4j.py

# Истражувачки notebook (онтологија, хиерархија, Венов дијаграм, мрежни прикази)
# (зависностите се инсталираат од првата ќелија во самиот notebook)
jupyter notebook notebooks/vezilka_graph.ipynb
```

## Структура

```
├── data/dijalekti/          <- корпус (metadata.json + training_data.jsonl)
├── ontology/
│   ├── vezilka.ttl          <- OWL онтологија (5 класи, 8 properties)
│   └── vezilka-shapes.ttl   <- SHACL shapes за валидација
├── scripts/
│   ├── extract_triples.py   <- pipeline за извлекување тројки
│   ├── validate.py          <- SHACL валидација
│   └── load_neo4j.py        <- вчитување во Neo4j Aura (property graph)
├── queries/
│   ├── q01-q08.rq           <- 8 SPARQL competency queries
│   └── run_queries.py       <- извршувач
├── notebooks/
│   └── vezilka_graph.ipynb  <- визуелизација (онтологија, хиерархија, Венов, мрежи)
├── .env.example             <- шаблон за Neo4j Aura credentials
└── output/                  <- генерирани фајлови (gitignored)
```

## Онтологија

**Класи:** Наречје, ДијалектнаГрупа, Дијалект, Лексема, Место

**Клучни properties:**

- `vez:appearsInDialect` - збор потврден во дијалект
- `vez:hasVariant` (symmetric) - cross-dialect варијантен пар
- `vez:dialectRegion` - дијалект -> географски регион (место)
- `skos:broader` - хиерархиска навигација низ наречје/група/дијалект

## Competency Questions

| # | Прашање |
|---|---------|
| Q1 | Кои зборови се карактеристични за одреден дијалект? |
| Q2 | Како се разликува лексиката меѓу два дијалекта? |
| Q3 | Кои зборови имаат варијанти во повеќе дијалекти? |
| Q4 | Колку зборови се заеднички за сите дијалекти? |
| Q5 | Кои се најфреквентните зборови по дијалект? |
| Q6 | Кои дијалекти се најслични според заеднички варијантни зборови? |
| Q7 | Кое наречје има најголема лексичка разновидност? |
| Q8 | Кои зборови варираат најмногу низ дијалектите? |

## Технологии

- Python + [rdflib](https://rdflib.readthedocs.io/) — RDF graph и SPARQL
- [pyshacl](https://github.com/RDFLib/pySHACL) — SHACL валидација
- OWL — онтолошки јазик
- SKOS — дијалектна хиерархија
- Turtle (.ttl) — RDF серијализација
